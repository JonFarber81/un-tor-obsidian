Use the Lore Table for **open-ended questions** about the world. Roll 1 **Feat Die** to determine the section, then 1 **Success Die** (d6) for the row. You'll get three words — **Action, Aspect, Focus** — use one, two, or all three as a prompt phrase.

> Roll 1 Feat Die (section) + 1 Success Die (row). Interpret the result in context of the current situation. If a result doesn't fit, roll again or check adjacent rows.
