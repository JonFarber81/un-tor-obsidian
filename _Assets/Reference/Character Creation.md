## Step 3 — Assign Attribute Scores

Each culture provides **three values** (see culture file) to assign freely among **Strength**, **Heart**, and **Wits**.

> [!Rolls] Key Formulas
> - **Attribute TN** = `18 − Attribute Score`
>   - Score 7 → TN 11 · Score 5 → TN 13 · Score 2 → TN 16
> - **Endurance** = Cultural base + Strength score (varies by culture)
> - **Hope** = 10 + Heart score (consistent across all cultures)
> - **Parry** = 10 + ranks in your primary weapon skill + shield bonus if any

### Derived Stat Examples

| Culture | Endurance Base | Example |
|---------|---------------|---------|
| Dwarves of Durin's Folk | 22 + STR | STR 7 → Endurance 29 |
| Hobbits of the Shire | 18 + STR | STR 2 → Endurance 20 |
| Men (most cultures) | ~20 + STR | See culture file |
| Elves | ~20 + STR | See culture file |

## Step 4 — Spend Starting Skill Points

Distribute **8 Skill Points** among any skills. **Maximum 3 ranks** in any single skill at character creation.

**Favoured Skills** for your culture start with **1 free rank** before you spend points. These are listed in each culture file.

### Skill List by Attribute

| :RaMuscleUp: Strength | :FasHeart: Heart | :FasBrain: Wits |
|-----------------------|-----------------|----------------|
| Awe | Enhearten | Persuade |
| Athletics | Travel | Stealth |
| Awareness | Insight | Scan |
| Hunting | Healing | Explore |
| Song | Courtesy | Riddle |
| Craft | Battle | Lore |

**Weapon Skills** (tracked separately — default 0):
Axes · Bows · Spears · Swords

> [!combat] Weapon Skill Tips
> At least 1–2 ranks in your primary weapon is important for survival. A rank of 3 is viable at start for a combat-focused character, but costs most of your 8 points.

## Step 6 — Starting Equipment

Your **Standard of Living** (from culture) determines starting gear quality and treasury.

| Standard | Starting Silver | Typical Gear |
|----------|----------------|-------------|
| Poor | 1d6 | Simple clothing, 1 basic weapon |
| Frugal | 2d6 | + basic travelling gear, dagger |
| Common | 3d6 | + 1 additional weapon or tool |
| Prosperous | 4d6 | + armour option, better weapons |
| Rich | 5d6 | + fine gear, horse/pony |

> Each culture also restricts certain gear — see culture files for weapon/armour limitations.
> See [[Weapons]] and [[Armor]] for all stat blocks.

## Advancement Reference

Advancement happens during the **Fellowship Phase** using points earned in the Adventure Phase.

| Spend | Cost | Effect |
|-------|------|--------|
| Skill Points | 1 per rank | Raise any skill |
| Adventure Points | 1 | Unlock a Cultural Virtue or Calling Virtue |
| Adventure Points | 2 | Raise Valour or Wisdom by 1 |
| Adventure Points | Varies | Raise an Attribute (see rulebook) |

> Cultural Virtues: see each culture file. Calling Virtues: see each calling file.
> Attributes may only be raised once per Fellowship Phase.
